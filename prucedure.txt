ML End-to-end Project :-

Steps :-

1]create a new git repo in github account.
2]create a new folder in your directory and copy that path.
3]open cmd prompt and change current path with copied path by typing :- cd <copied path>
4]now to open vs code directly from cmd prompt in the project directory type :- code .
5]now in vs code to create new invironment open teminal and type :- conda create -p <env name> -y
6]to activate new env type :- conda activate <env name>/
7]to add repository type :- git init
8]create a README.md file in project and add this file in git repo type :- git add README.md and to commit changes 
   type :- git commit -m "<type msg in double quotes only>" and to see status type :- git status
9]to check out branch type :- git branch -M main
10]to add origin type :- git remote add origin https://github.com/AJ2408/mlproject.git
11] to check type :- git remote -v  then we will get fetch and push links.
12]git configuration :- for username -> git config --global user.name  for email --> git config --global user.email
     and to change user name or email just type in double quotes new name or email after git config --global user.name/email " "
13]now push the code or files to main branch by typing :- git push -u origin main
14]now create .gitignore file in github repo and select python template and commit chage. 
     Now to see updations in project type :- git pull
15]now create setup.py and requirements.txt files in project.
16]create folder source and in it create __init__.py file.
17]write <-e .> in requirements.txt file to automatically triggered setup.py.
18]now run command :- pip install -r requirements.txt
19]now type commands :- < git add . > to add files ,< git commit -m "msg" > to commit changes and < git push -u origin main > to     push file to repo.
